Eng2Te
======

* A simple Artha-Inspired, RegEx supporting, English to Telugu Dictionary written in pyGtk. 
* Resources From: http://sourceforge.net/projects/tel-dictionary/


How To Run
----------
  * GUI -> Run gui.py
      
          $python gui.py
    
  * CLI -> Run test.py
          
          $python test.py word1 word2 regex1
